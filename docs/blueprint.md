# **App Name**: Post Weaver

## Core Features:

- Text Input: Allow users to input their article or text.
- Automated Text Chunking: Automatically split the text into logical paragraphs, each 250-350 characters long. If a title is absent, the system will make the first sentences of first paragraph into a title.
- Template Selection: Enable users to pick from flat color or gradient background templates.
- Font Selection Tool: Let users select fonts (Plus Jakarta Sans, Playfair Display → Inter, and Special Elite).
- Design preview: Enable users to visualize a horizontal slider presenting ready-to-publish JPG images.
- Adaptive Contrast: The system uses a tool to ensure high readability with proper color contrast, even as background and font color change. The tool proposes 5 different color combination schemes.
- Download Designs: Users download selected designs as JPG files.

## Style Guidelines:

- Primary color: Saturated blue (#4285F4), evokes trustworthiness.
- Background color: Light grayish-blue (#E8F0FE), supports readability.
- Accent color: A warm, desaturated yellow (#F7DE79), offers a contrast.
- Font selection includes: 'Plus Jakarta Sans' (sans-serif), 'Playfair' (serif for titles) → 'Inter' (sans-serif for body), and 'Special Elite'.
- Note: currently only Google Fonts are supported.
- Text aligns to the left with 10px side margins, designs have 20px inner spacing.
- The 1080x1350 designs populate a slider. Designs remain consistent.
